# tree-sitter-language-pack - Elixir Bindings

Pre-compiled tree-sitter grammars for 305 programming languages

## Installation

Add `:tree_sitter_language_pack` to your `mix.exs` dependencies:

```elixir
defp deps do
  [
    {:tree_sitter_language_pack, "~> 1.8.0-rc.26"}
  ]
end
```

## Quick Start

```elixir
Tree_sitter_language_pack.hello()

# See https://github.com/kreuzberg-dev/tree-sitter-language-pack for usage examples.
```
