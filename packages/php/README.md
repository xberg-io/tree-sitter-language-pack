# tree-sitter-language-pack - PHP Bindings

Pre-compiled tree-sitter grammars for 305 programming languages

## Installation

```bash
composer require kreuzberg-dev/tree-sitter-language-pack
```

## Quick Start

```php
<?php

use ts_pack_core_php;

// See https://github.com/kreuzberg-dev/tree-sitter-language-pack for usage examples.
```
